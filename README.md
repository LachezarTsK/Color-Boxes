

The application generates random-color boxes. Depending on the selected choice, the number of boxes can be between 10 and 100. The default value upon start of the application is 20 boxes. In addition, upon clicking on a specific box, its color is changed by another random color.
Of course, this application does not directly address some specific problem or need. However, in one form, or another, the application could be partof some real world project, so mastering it is worth the effort.
To run the application, click on: https://lachezartsk.github.io/Color-Boxes/
